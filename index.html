Quantum Manifold Learning

Interactive visualization of low-dimensional manifolds discovered in high-dimensional quantum datasets using diffusion coordinates.

Project Overview

Quantum systems live in extremely high-dimensional Hilbert spaces. However, physical states often lie on low-dimensional manifolds embedded within those spaces.

This project applies unsupervised machine learning techniques to uncover that intrinsic geometry using diffusion maps applied to classical shadow measurements of quantum states.

Methodology

  • Generate quantum state measurements using classical shadows
  • Construct similarity kernel between quantum states
  • Apply diffusion map embedding
  • Extract 3–5 diffusion coordinates
  • Visualize the resulting manifold geometry

Results

The analysis demonstrates that the structure of the quantum system can be represented using only a few diffusion coordinates. This reduced representation captures most of the relationships between states while enabling geometric visualization of the quantum state space.
