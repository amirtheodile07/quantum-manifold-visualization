01 — The Problem

Quantum states live in exponentially large spaces

A system of n qubits exists in a Hilbert space of dimension 2ⁿ. For 50 qubits that's over 10¹⁵ dimensions — far beyond any direct analysis. Yet physical systems don't freely roam this space. They're constrained to thin manifolds shaped by the underlying Hamiltonian.

The question is: can we find those manifolds without ever working in the full high-dimensional space?

What is a quantum manifold? +

A manifold is a smooth, lower-dimensional surface embedded in a higher-dimensional space — like a 2D sheet crumpled inside 3D space. Quantum states parameterized by physical fields (like h₁, h₂) naturally form such surfaces, since nearby parameter values produce nearby states.

Why does this matter? +

If we can identify the manifold, we can classify quantum phases, predict observables from geometry alone, and compress exponentially large datasets into a handful of coordinates — enabling efficient quantum simulation and machine learning.

02 — Data Collection

Classical shadows of quantum states

We can't store a full quantum state — but we can take randomized snapshots of it. The classical shadow protocol applies random unitaries and measures in the computational basis, producing a compact classical description that statistically encodes the state's properties.

From these shadows we extract pairwise similarity scores between quantum states, forming a kernel matrix that captures the geometry of state space without ever constructing the full Hilbert space representation.

How classical shadows work +

Apply a random unitary U to the state, measure in the computational basis to get a bitstring b, then store the "shadow" U†|b⟩⟨b|U. Averaging over many such snapshots lets you estimate any low-weight observable efficiently — far fewer measurements than full tomography.

The String Order Parameter (SOP) +

The SOP is a nonlocal observable that distinguishes topological from trivial phases. It takes large values in the topological phase and near-zero values in the trivial phase, making it the key quantity we aim to predict from diffusion coordinates alone.

03 — Dimensionality Reduction

Diffusion maps reveal intrinsic geometry

The kernel matrix defines a diffusion process over the data. Eigenvectors of this diffusion operator — the diffusion coordinates — parameterize the manifold in order of geometric importance. The first few coordinates capture the dominant structure.

We found that coordinates 1 through 3 encode the phase transition boundary, the strength of ordering, and secondary variations — together sufficient to predict the String Order Parameter with R² ≈ 0.80.

Why not PCA or t-SNE? +

PCA finds linear structure only. t-SNE is nonlinear but distorts global geometry and has no out-of-sample extension. Diffusion maps preserve the true geodesic distances along the manifold and produce coordinates with direct physical meaning — each one corresponds to a mode of variation in the quantum system.

What does the regression test? +

A linear regression from diffusion coordinates to the SOP tests whether the manifold geometry is physically meaningful. If R² is high, the geometric structure — discovered with no knowledge of the physics — nonetheless encodes the physics. R² ≈ 0.80 confirms this strongly.

05 — About

About this work

This project was developed as part of undergraduate research in quantum machine learning at Binghamton University, in collaboration with faculty and graduate students.

The core contribution is demonstrating that unsupervised geometric methods — requiring no labeled data or prior knowledge of the physics — can discover representations of quantum state space that are both compact and physically interpretable.

The diffusion map embedding is computed from pairwise state similarities derived from classical shadow measurements, then evaluated by its ability to linearly predict the String Order Parameter across the phase diagram.
